ergo status
list all branches with no of commits behind and ahead of base branch

ergo pending
list detail commits behind base branch

ergo draft-release --dry-run
shows text of the release to be created from base branch to target branch

ergo draft-release
creates github release from base branch to target branch

ergo deploy
pushes base branch to target branches

ergo create-pull
cerates a pull request from base branch to target branch
